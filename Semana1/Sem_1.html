Semana 1: Introducción al curso

¿Qué hemos aprendido?


El curso abarcara el desarrollo integral de aplicaciones web, desde el frontend hasta el backend. En el frontend, se enseñaron los fundamentos de HTML, CSS (incluyendo Bootstrap y Tailwind), y JavaScript, así como el uso de React. Para el backend, se trabajó con PHP, JSP, Python y Django, enfocándose en la creación de APIs RESTful y microservicios, e implementando autenticación, validación y consumo de datos. También se abordaron principios de arquitectura web moderna, manipulación del DOM, manejo de eventos, hooks y técnicas avanzadas de estilizado.

Objetivos de Aprendizaje

  • Desarrollar soluciones fullstack con integración de frontend y backend usando APIs.
  • Diseñar interfaces efectivas y funcionales para usuarios.
  • Desarrollar aplicaciones web frontend de tipo SPA (Single Page Application).

Tecnologías Utilizadas

  • Frontend: HTML5, CSS3, Bootstrap, Tailwind CSS, JavaScript, TypeScript, React.
  • Backend: PHP, JSP, Python, Django.
  • Herramientas adicionales: Visual Studio Code, Axios, GitHub, Docker, Kubernetes, Microsoft Teams.

Conclusión

El curso ofrecerá una formación integral en el desarrollo web moderno, preparando al estudiante para enfrentar los desafíos del mercado laboral actual. Se promueve el uso de frameworks robustos, arquitectura escalable y herramientas actualizadas, todo bajo un enfoque práctico, ético y colaborativo. Se logra una comprensión sólida del ciclo completo de desarrollo de aplicaciones web.

Reflexión

Este curso no solo ampliará mis conocimientos técnicos, sino que también fortaleció mi capacidad para resolver problemas reales a través del desarrollo de soluciones web eficientes. La experiencia práctica con herramientas actuales y casos reales me permitió visualizar con claridad el impacto del desarrollo web en la sociedad, preparándome para contribuir de manera significativa en el ámbito profesional.
