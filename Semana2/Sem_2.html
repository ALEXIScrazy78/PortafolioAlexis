Semana 2: Uso de Visual Studio Code

¿Qué hemos aprendido?


Esta semana, hemos dominado el uso eficiente de Visual Studio Code. Centramos nuestra práctica en Emmet, un plugin que agiliza la creación de código HTML con abreviaciones, y en los atajos de teclado de VSCode, herramientas clave para aumentar la productividad en el desarrollo.

Objetivos de Aprendizaje

  • Dominar la interfaz y configuración básica de VS Code.
  • Aprender a usar Emmet para generar código HTML
  • Aplicar atajos de teclado de VS Code para optimizar el flujo de trabajo

Tecnologías Utilizadas

  • Visual Studio Code
  • Atajos de teclado nativos de VS Code

Conclusión

El uso de codigos Emmet, atajos de teclado y otras herramientas en Visual Studio Code mejoran la eficiencia de como podemos escribir el código. Practicar cada una de estas nuevas habilidades adquiridas, pueden ser relevantes en el mundo laboral donde el tiempo y la calidad son uno de los activos mas preciados.

Reflexión

Seguiré perfeccionando mi uso de Visual Studio Code mediante la práctica constante y el aprovechamiento de herramientas como Emmet, lo que me permitirá desarrollar proyectos web con mayor rapidez, precisión y eficiencia.
