Semana 7: React, Componentes y Hooks Avanzados

¿Qué hemos aprendido?


He aprendido a construir interfaces web usando componentes funcionales en React y a controlar la lógica del estado y del ciclo de vida con hooks como useState y useEffect.

Objetivos de Aprendizaje

  • Implementar formularios controlados con validación.
  • Utilizar `useEffect` para efectos secundarios como la carga de datos.
  • Aplicar el Context API para la gestión global del estado.
  • Comprender patrones de componentes más complejos.

Tecnologías Utilizadas

  • React (JSX, Hooks)
  • HTML5
  • CSS3
  • JavaScript ES6+
  • Visual Studio Code

Ejercicios de React (Avanzados)

1. Formulario con Validación y Manejo de Estado

Un ejemplo que demuestra cómo manejar entradas de usuario, validar formularios y gestionar el estado complejo.

Ver Ejercicio 1

2. Carga de Datos con `useEffect`

Muestra cómo utilizar el hook `useEffect` para simular la carga de datos de una API y manejar estados de carga y error.

Ver Ejercicio 2

Conclusión

La inmersión en componentes y hooks de React ha sido transformadora. Ahora entiendo cómo construir aplicaciones más robustas y con una lógica de estado más clara, lo cual es fundamental para proyectos a gran escala.

Reflexión

Dominar hooks como `useState` y `useEffect` ha simplificado enormemente la forma en que manejo el estado y los efectos secundarios, haciendo mi código más legible y eficiente. El Context API es una revelación para evitar el "prop drilling".
