Semana 4: CSS Bootstrap y Tailwind

¿Qué hemos aprendido?


He aprendido a construir interfaces web utilizando los frameworks CSS Bootstrap y Tailwind, entendiendo las diferencias entre el enfoque de componentes predefinidos de Bootstrap y el enfoque de clases utilitarias de Tailwind. Ahora puedo aplicar lo básico de manera, ordenada y visualmente atractivas de forma eficiente.

Objetivos de Aprendizaje

  • Comprender la estructura y propósito de los frameworks Bootstrap y Tailwind.
  • Utilizar el sistema de rejilla (grid) de Bootstrap para crear layouts responsivos.
  • Aplicar estilos mediante CSS usando selectores y propiedades.
  • Comparar ventajas y desventajas entre ambos frameworks.

Tecnologías Utilizadas

  • Bootstrap 5
  • Tailwind CSS
  • HTML5
  • Visual Studio Code

Conclusión

Tanto Bootstrap como Tailwind CSS son herramientas poderosas que facilitan y aceleran el desarrollo web moderno. Bootstrap destaca por su facilidad y velocidad gracias a componentes listos para usar, mientras que Tailwind ofrece una personalización detallada ideal para diseños únicos.

Reflexión

Explorar estos frameworks me hizo ver que no hay una única manera de diseñar una página web. Bootstrap me ayudó a construir rápidamente prototipos funcionales, mientras que Tailwind me ofreció control absoluto sobre el diseño. Aprender ambos me brinda una ventaja competitiva al momento de elegir la herramienta más adecuada según el proyecto.
