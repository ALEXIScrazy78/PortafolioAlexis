Semana 5: Javascript

¿Qué hemos aprendido?


He aprendido a utilizar JavaScript para dar vida a las páginas web, desde lo más básico como variables y funciones, hasta manipular el DOM, gestionar eventos y aplicar lógica dinámica. Entendí cómo JavaScript se integra con HTML y CSS para construir interfaces interactivas y cómo usarlo para comunicarme con servidores mediante APIs.

Objetivos de Aprendizaje

  • Comprender la sintaxis básica de JavaScript (variables, operadores, funciones).
  • Aplicar estructuras de control como condicionales y bucles.
  • Manipular elementos del DOM (agregar, quitar o cambiar contenido).
  • Realizar solicitudes a servidores mediante fetch o async/await.

Tecnologías Utilizadas

  • JavaScript
  • HTML5
  • CSS3
  • Visual Studio Code
  • Google Chrome DevTools

Conclusión

JavaScript es la herramienta clave para hacer que los sitios web pasen de ser estáticos a dinámicos e interactivos. Su versatilidad lo convierte en el lenguaje más utilizado del desarrollo web moderno. Esta clase permitió sentar las bases sólidas para avanzar hacia frameworks como React, Vue o Angular.

Reflexión

Aprender JavaScript me abrió las puertas al mundo real del desarrollo web. Comprendí que escribir código no solo se trata de que funcione, sino de cómo se comporta en diferentes escenarios. JavaScript me desafió a pensar de manera lógica, resolver errores y comprender cómo los usuarios interactúan con las páginas.
