Semana 6: Fundamentos de React

¿Qué hemos aprendido?


He aprendido a construir interfaces web dinámicas y modulares usando React, comprendiendo la lógica de los componentes, el uso de props y estados, y cómo React actualiza el DOM de manera eficiente.

Objetivos de Aprendizaje

  • Comprender los conceptos fundamentales de React (componentes, JSX, props, state).
  • Crear componentes funcionales y usar Hooks básicos como `useState`.
  • Manejar eventos en React.
  • Renderizar listas y aplicar renderizado condicional.

Tecnologías Utilizadas

  • React (JSX)
  • HTML5
  • CSS3
  • JavaScript ES6+
  • Visual Studio Code

Conclusión

React es una biblioteca poderosa para construir interfaces de usuario, enfocándose en la composición de componentes y la gestión eficiente del DOM. Comprender sus fundamentos es crucial para el desarrollo front-end moderno.

Reflexión

La transición a React me ha permitido ver el desarrollo web desde una perspectiva más modular y declarativa. Me ha enseñado la importancia de la gestión del estado y cómo construir aplicaciones escalables y fáciles de mantener.
