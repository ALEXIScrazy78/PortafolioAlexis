Semana 3: Fundamentos de HTML y CSS

¿Qué hemos aprendido?


He aprendido a estructurar contenido web utilizando etiquetas HTML y a aplicar estilos visuales con CSS para lograr interfaces ordenadas, estéticas y adaptables. Comprendí la diferencia entre estructura (HTML) y estilo (CSS), y cómo ambas trabajan juntas para construir sitios web funcionales y visualmente agradables.

Objetivos de Aprendizaje

  • Comprender la estructura básica de un documento HTML.
  • Conocer y usar etiquetas semánticas HTML5.
  • Aplicar estilos mediante CSS usando selectores y propiedades.
  • Diferenciar entre estilos inline, internos y externos.

Tecnologías Utilizadas

  • HTML5
  • CSS3
  • Visual Studio Code
  • Google Chrome
  • Extensiones VSC

Ejercicio 1: Galería de Imágenes Interactiva (CSS Grid y Hover)

Una galería de imágenes responsiva que utiliza CSS Grid para el diseño y efectos de escala y superposición en el hover, demostrando interacción solo con CSS.

Título de Imagen 2

Título de Imagen 3

Título de Imagen 4

<!-- HTML para la galería -->
<div class="gallery-container">
    <div class="gallery-item">
        <img src="path/to/image1.jpg" alt="Imagen 1">
        <div class="gallery-overlay">
            <h4>Título de Imagen 1</h4>
            <p>Breve descripción.</p>
        </div>
    </div>
    <!-- Repite para más imágenes -->
</div>

<!-- CSS relevante (incluido en el head del HTML para este ejemplo) -->
.gallery-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;
}
.gallery-item {
    position: relative;
    overflow: hidden;
    border-radius: 8px;
    box-shadow: 0 4px 10px rgba(0,0,0,0.1);
}
.gallery-item img {
    width: 100%;
    height: 200px;
    object-fit: cover;
    display: block;
    transition: transform 0.3s ease-in-out;
}
.gallery-overlay {
    /* ... estilos para la superposición ... */
    opacity: 0;
    transition: opacity 0.3s ease-in-out;
}
.gallery-item:hover img {
    transform: scale(1.1);
}
.gallery-item:hover .gallery-overlay {
    opacity: 1;
}

Ejercicio 2: Ventana Modal Solo con CSS

Este ejercicio demuestra cómo crear una ventana modal (pop-up) que se abre y cierra utilizando únicamente HTML y CSS, sin necesidad de JavaScript. Se emplea la funcionalidad de un checkbox oculto para controlar la visibilidad.

¡Bienvenido al Modal CSS!

Este es un ejemplo de una ventana modal que se abre y cierra solo con HTML y CSS. ¡Es increíble lo que se puede lograr sin JavaScript!

<!-- HTML para el modal -->
<label for="modal-toggle" class="modal-trigger">Abrir Modal</label>

<input type="checkbox" id="modal-toggle" class="modal-toggle">
<div class="modal-content-wrapper">
    <div class="modal-box">
        <h2>¡Bienvenido al Modal CSS!</h2>
        <p>Este es un ejemplo de una ventana modal...</p>
        <label for="modal-toggle" class="modal-close-button">&times;</label>
    </div>
</div>

<!-- CSS relevante (incluido en el head del HTML para este ejemplo) -->
.modal-toggle {
    display: none; /* Oculta el checkbox */
}
.modal-content-wrapper {
    position: fixed;
    /* ... estilos para centrar y ocultar ... */
    visibility: hidden;
    opacity: 0;
    transition: visibility 0s, opacity 0.3s ease-in-out;
}
.modal-toggle:checked + .modal-content-wrapper {
    visibility: visible;
    opacity: 1;
}
.modal-box {
    /* ... estilos para el contenido del modal ... */
    transform: translateY(-50px);
    opacity: 0;
    transition: transform 0.3s ease-out, opacity 0.3s ease-out;
}
.modal-toggle:checked + .modal-content-wrapper .modal-box {
    transform: translateY(0);
    opacity: 1;
}

Ejercicio 3: Fondo Gradiente Animado con Texto

Una sección visualmente atractiva con un fondo de gradiente que se anima suavemente en un ciclo infinito, combinado con texto estilizado que se superpone de manera legible.

¡Diseño Web Moderno!

Descubre el poder de CSS para crear experiencias visuales impactantes y fluidas.

<!-- HTML para la sección -->
<div class="animated-gradient-section">
    <div>
        <h3>¡Diseño Web Moderno!</h3>
        <p>Descubre el poder de CSS...</p>
    </div>
</div>

<!-- CSS relevante (incluido en el head del HTML para este ejemplo) -->
.animated-gradient-section {
    height: 300px;
    background: linear-gradient(45deg, #FF6600, #17408B, #913b01, #000);
    background-size: 400% 400%;
    animation: gradientAnimation 15s ease infinite;
    display: flex;
    justify-content: center;
    align-items: center;
    color: white;
    text-align: center;
    border-radius: 10px;
    position: relative;
    overflow: hidden;
}

.animated-gradient-section h3 {
    font-size: 2.5rem;
    text-shadow: 2px 2px 5px rgba(0,0,0,0.5);
}

.animated-gradient-section p {
    font-size: 1.2rem;
    text-shadow: 1px 1px 3px rgba(0,0,0,0.5);
}

@keyframes gradientAnimation {
    0% { background-position: 0% 50%; }
    50% { background-position: 100% 50%; }
    100% { background-position: 0% 50%; }
}

Conclusión

El dominio de HTML y CSS constituye la base esencial para cualquier desarrollador web. Esta clase permitió sentar las bases para construir páginas bien estructuradas y visualmente agradables, lo que facilita avanzar posteriormente hacia tecnologías más complejas como JavaScript o frameworks como React.

Reflexión

Iniciar con HTML y CSS fue una experiencia reveladora, ya que me permitió ver cómo pequeños bloques de código se transforman en interfaces reales. Me di cuenta de que el diseño web no solo implica estética, sino también accesibilidad, usabilidad y rendimiento. Esta base es crucial para cualquier desarrollo posterior.
